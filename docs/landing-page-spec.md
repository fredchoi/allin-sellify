# 셀러 올인원 SaaS — 랜딩페이지 구성안

> 마케팅팀 기획 문서 v1.0 | 2026-03-28

---

## 1. 페이지 목표

| 항목 | 내용 |
|------|------|
| 주요 목표 | 무료 체험 가입 전환 (CTA: "14일 무료 체험") |
| 보조 목표 | 서비스 인지도 확보, 데모 영상 조회 |
| 타겟 페르소나 | 네이버/쿠팡 셀러 (월 매출 100만~5,000만 원대, 1인~소규모 운영) |
| 핵심 메시지 | "상품 소싱부터 정산까지, AI가 자동으로" |
| 톤 앤 매너 | 신뢰감 + 실용적 + 깔끔한 SaaS 톤. 과장 없이 숫자로 증명 |

## 2. 섹션별 구성

### Section 01 — 히어로 (Hero)

```
┌─────────────────────────────────────────────────────────────────┐
│  [로고]                    기능  요금제  고객후기  FAQ   [로그인] │
│                                                                 │
│         상품 소싱부터 정산까지,                                    │
│         AI가 전부 해드립니다.                                     │
│                                                                 │
│    도매 상품 수집 → AI 가공 → 멀티마켓 등록 → 재고·주문·정산      │
│    셀러가 잠든 사이에도 매출은 올라갑니다.                          │
│                                                                 │
│    [14일 무료 체험 시작하기]  [3분 데모 영상 보기 ▶]              │
│                                                                 │
│    ✓ 카드 등록 없이 시작  ✓ 설치 불필요  ✓ 5분 만에 첫 상품 등록  │
│                                                                 │
│         [ 대시보드 스크린샷 / 제품 데모 애니메이션 ]                │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**카피 포인트:**
- 헤드라인: 구체적 결과 약속 (소싱→정산 전 과정)
- 서브 헤드라인: 자동화 강조 ("잠든 사이에도")
- 마이크로카피: 가입 장벽 제거 (카드 없음, 설치 없음)
- 비주얼: 실제 대시보드 UI를 보여주어 신뢰감 형성

### Section 02 — 셀러의 현실 (Pain Point)

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│            혹시 이런 상황, 익숙하지 않으세요?                       │
│                                                                 │
│  ┌───────────┐  ┌───────────┐  ┌───────────┐  ┌───────────┐   │
│  │ 😩        │  │ 😤        │  │ 😰        │  │ 😵        │   │
│  │ 상품 하나   │  │ 네이버·쿠팡 │  │ 재고 0인데  │  │ 정산 끝나면 │   │
│  │ 등록하는 데 │  │ 따로따로    │  │ 주문이     │  │ 남는 게    │   │
│  │ 40분       │  │ 올리기     │  │ 들어옴     │  │ 있는 건지  │   │
│  └───────────┘  └───────────┘  └───────────┘  └───────────┘   │
│                                                                 │
│         → 이 모든 문제, 하나의 플랫폼에서 해결됩니다.               │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**의도:** 타겟 셀러가 "나 얘기다"라고 느끼게 만드는 공감 섹션. 이후 솔루션으로 자연스럽게 연결.

### Section 03 — 핵심 파이프라인 (How It Works)

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│          클릭 몇 번이면 상품이 매출이 됩니다                        │
│                                                                 │
│   ① 수집          ② AI 가공        ③ 등록          ④ 자동화     │
│  ┌────────┐     ┌────────┐      ┌────────┐      ┌────────┐    │
│  │ 도매꾹  │ ──→ │ AI 제목 │ ──→  │ 네이버  │ ──→  │ 재고    │    │
│  │ 도매토피아│     │ 후킹문구 │      │ 쿠팡   │      │ 주문    │    │
│  │ 트렌드   │     │ 썸네일   │      │ 자사몰  │      │ 배송    │    │
│  │ 키워드   │     │ 옵션정리 │      │ 원클릭  │      │ 정산    │    │
│  └────────┘     └────────┘      └────────┘      └────────┘    │
│    자동 수집        4초 완료        동시 등록       24시간 운영    │
│                                                                 │
│                                                                 │
│   ⑤ 마케팅         ⑥ 분석                                       │
│  ┌────────┐     ┌────────┐                                      │
│  │ 블로그  │     │ 키워드  │                                      │
│  │ SNS    │     │ 매출    │                                      │
│  │ AI 생성 │     │ 마진    │                                      │
│  │ 예약발행 │     │ 트렌드  │                                      │
│  └────────┘     └────────┘                                      │
│   자동 콘텐츠      데이터 인사이트                                  │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**카피 포인트:**
- 6단계를 시각적으로 간결하게 표현
- 각 단계마다 한 줄 베네핏 ("4초 완료", "동시 등록", "24시간 운영")
- 스크롤 시 각 단계가 순차적으로 애니메이션 등장 (인터랙션)

## 5. 전환 최적화 체크리스트

- [ ] CTA 버튼: 스크롤 위치와 무관하게 항상 접근 가능 (Sticky 네비게이션 또는 플로팅)
- [ ] 히어로 CTA 2개: 주요 (무료 체험) + 보조 (데모 영상)
- [ ] 페이지 중간 CTA 반복: Section 04 이후, Section 07 (요금제), Section 10 (최종)
- [ ] 모바일 최적화: 터치 타겟 44px 이상, CTA 버튼 풀 너비
- [ ] 로딩 속도: LCP 2.5초 이내 (이미지 WebP, 폰트 preload, 코드 스플릿)
- [ ] 소셜 프루프 배치: 숫자 → 후기 → 요금제 순서 (신뢰 → 확신 → 결제)
- [ ] Exit Intent 팝업: 이탈 시 "무료 체험" 리마인드 (데스크톱 전용)
- [ ] A/B 테스트 대상: 히어로 헤드라인, CTA 문구, 요금제 레이아웃
